# BWD3 : Projet Strata

## Etape 1 : Setup
1. Déziper le fichier "sourceDuProjet.zip".
1. Le positionner dans votre espace de travail.
1. Fin de l’installation.

## Etape 2 : Ajouter des variables PHP
1. Renommer votre fichier "index.html" en "index.php".
1. Créer un fichier "config.php".
1. Inclure le fichier "config.php" en entête de votre fichier "index.php".
1. Ajouter les variables (dans le fichier "config.php"), et les initialiser avec des valeurs :
    * $twitter = "";
    * $github = "";
    * $dribbble = "";
    * $email = "";
    * $adresse = "";
    * $ville = "";
    * $cp = "";
    * $tel = "";

## Etape 3 : Soumettre un formulaire
1. Soumettre le formulaire et afficher les valeurs inscritent (var_dump).

## Etape 4 : Utiliser une boucle
1. Créer un tableau PHP, avec 4 "title", "description" et "urlImage".
1. Boucle "foreach" pour afficher les nouveaux "Work".
